HOW IT WORKS

五個步驟，做出你的編輯器

全程不需要懂技術。你只需要一個 Claude 帳號和幾分鐘時間。

1
複製下方的啟動提示詞
頁面下方有一段完整的提示詞，點「複製全部」即可。
2
打開 Claude Chat，貼上去
前往 chat.claude.ai，開新對話，貼進去送出。
3
上傳參考圖（可選）
Claude 會先問你有沒有參考風格的圖片。有的話直接上傳截圖，它會先分析風格再問細節。沒有也沒關係，用文字描述就好。
4
回答 Claude 的問題
Claude 會一次問你一個問題——帳號名稱、品牌風格、輪播結構、要不要放照片…約 7 個問題，用中文回答就好。
5
存成 HTML，用 Chrome 打開
把 Claude 輸出的程式碼全部複製，貼到記事本，另存成 .html 檔，用 Chrome 或 Safari 打開就是你的專屬編輯器。
STARTER PROMPT

啟動提示詞

把以下內容全部複製，貼進 Claude Chat 對話框送出即可啟動。
Claude 會主動引導你完成所有設定。

你現在是一個專業的【輪播貼文編輯器製作助手】。 你的任務是：先收集我的需求，再幫我生成一個完整的、可以在瀏覽器直接使用的 HTML 輪播貼文編輯器。 請用友善的繁體中文跟我對話，一次只問一個問題。 --- 【第一步：先問我有沒有參考圖】 請先問我：「你有沒有想參考的風格圖片？可以是你喜歡的帳號截圖、你常看到的版型、或任何你覺得『我想要類似這種感覺』的圖。有的話先上傳，沒有也沒關係，我們直接用文字描述。」 如果我上傳了圖片，請先仔細分析圖片的配色、版型結構、整體風格感受，告訴我你觀察到什麼，再根據觀察繼續問細節。 如果我沒有上傳圖片，直接進入第二步。 --- 【第二步：依序問以下問題，一次一個】 問題 1：你的帳號名稱或品牌名稱叫什麼？（會自動加在每張圖的浮水印上） 問題 2：你的品牌給人什麼感覺？用幾個詞形容。（例如：高級冷靜、溫暖療癒、活潑可愛、神秘靈性、簡約知性） 問題 3：你偏好深色背景還是淺色背景？有沒有喜歡的顏色或色碼？ 問題 4：你希望重點文字（強調字）是什麼顏色？如果有第二個常用強調色也一起說。 問題 5：你的輪播通常做幾張？每一張大概放什麼內容？（例如：5 張，第 1 張封面、第 2-4 張正文、第 5 張 CTA） 問題 6：你會在圖片上放背景照片嗎？（需要的話，我會幫你加入照片上傳、位置調整、明暗控制的功能） 問題 7：你想要以下哪些選填欄位？（可以多選，不要也沒關係） - 品牌介紹文字欄（放一段固定的品牌說明） - Hashtag 欄（固定 hashtag 存在這裡） --- 【問完之後，直接根據我的答案生成 HTML，不需要再確認，直接輸出完整程式碼】 這個 HTML 編輯器必須包含以下所有功能： 整體架構：三欄版面（左側縮略圖導航、中間編輯區、右側即時預覽），左側縮略圖可點擊切換頁面並顯示頁碼。 文字欄位（每頁都有）：標籤欄（Badge）、標題欄（支援多行 textarea，預覽時完整保留換行）、副標題欄、內文欄（多行）。最後一頁額外有 CTA 欄。 選填欄位：依問題 7 決定是否加入品牌介紹文字欄和 Hashtag 欄。 內嵌排版語法（在任何文字欄都可以使用）： 1. **文字** → 主強調色文字 2. ##文字## → 第二強調色文字 3. --- → 橫向分隔線 4. [[ 和 ]] 包住的文字區塊 → 半透明底色攻略卡片，裡面每行以「- 」開頭自動變條列項目 照片功能：依問題 6 決定是否加入（上傳照片、水平/垂直位置滑桿、縮放、模糊、暗化）。 儲存：換頁時自動儲存（localStorage），頁面上方有「載入已儲存」和「重置」按鈕。 浮水印：每張圖右下角自動顯示帳號名稱 + 頁碼（例如 1/5）。 匯出：下載單張 PNG（2x 解析度）、一鍵下載全部 PNG 打包成 ZIP（JSZip + html2canvas，ZIP 以帳號名稱命名）。 設計：顏色、字體、風格完全根據我的品牌答案或參考圖設計，攻略卡片的半透明底色要跟整體配色協調，不要用通用模板。 輸出格式：完整可用的單一 HTML 檔案，所有功能內嵌，無需後端。
SYNTAX GUIDE

內嵌排版語法

做好編輯器後，在任何文字欄裡輸入這些符號，預覽自動轉換樣式——每一頁都能用，不需要進任何設定。

**重要文字**
→ 顯示為品牌主強調色文字，用來突出最重要的訊息
##另一種重點##
→ 顯示為第二強調色文字，區分不同層次的重點
---
→ 橫向分隔線，把段落切開，增加視覺層次
[[ 內容 ]]
→ 半透明底色攻略卡片。裡面每行以 - 開頭自動變條列。想用就包，不影響其他頁面。
✦ 攻略卡片完整寫法
[[
- 第一個要點
- 第二個要點
- 第三個要點
]]

這整塊在預覽裡會變成一個有半透明背景的卡片，自動條列顯示。

做出來之後想調整怎麼辦

第一版不完美很正常。在同一個 Claude 對話繼續說就好。

顏色
「背景色太亮，改成更深的 #1a1a2e」
字體
「標題字體改大一點，大約 24px」
版面
「三欄版面跑掉了，幫我修好」
卡片
「[[ ]] 攻略卡片的底色太深，調淡一點，邊框加一條細線」
解析度
「下載的圖片太模糊，把匯出解析度改成 3x」
FAQ

常見問題

存成 HTML 之後打開是空白的？ +
確認副檔名是 .html（不是 .html.txt）。Windows 可能需要先開「顯示副檔名」才看得到。改好後用 Chrome 或 Edge 打開。
[[ ]] 語法打了但預覽沒有變化？ +
確認 [[ 和 ]] 各自獨立在一行，中間每個條目以 - （減號加空格）開頭。
可以用中文字體嗎？ +
可以。跟 Claude 說你想要的字體風格，或直接給字體名稱（例如：思源黑體、Noto Sans TC、金萱），它會從 Google Fonts 自動引入。
做好之後還可以繼續修改嗎？ +
可以。把 HTML 檔案的程式碼貼給 Claude，說「我想調整這裡」，它就能繼續修改。建議每次修改後都存一個新版本備份。
